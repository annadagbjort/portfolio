The Project

Description

The focus of this project was to design a character and a background to create a sprite animation.

I chose Adventure Time as my inspiration and designed the character “Pjási”. I then created a background with green grass and a blue sky and finally made a sprite animation of him riding a hobby horse.

I learned about the different animation principles and creating a walk-cycle for a character. I gained a lot of experience in Illustrator in this project which I enjoyed.
