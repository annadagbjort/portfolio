The Project

Description

The focus of this project was to create an onboarding through gamification. We chose to introduce what extra-curricular programs KEA has to offer.

In the project I drew all the characters in illustrator, created the moodboard, created the buttons and speech bubbles, wrote the conversations and my teammate and I made the final version of the storyboard together.

This was the first time I created anything with gamification which was a fun new experience.
