The Project

Description

In this project the main focus was on the user experience. We learned how important it is to do a lot of research when the goal is to create a good user experience. We did a desk research before deciding on anything about the website. In the theme we also learned about UI kits and how to use them to save time and get new ideas. We then used UI kits when designing the website which gave us more time to think about how we could create an optimal user experience. We conducted a “Think a loud” test which was very helpful for us e.g. we made it clearer in the sign-up form which information we needed. In the form people also got confused why we asked for country and age and that is why we added a question mark for people to learn about what we use this information for.
